# COMP3005-A3Q1

Before running the application, first create a database in PostgreSQl. Then run "COMP3005 A3Q1 ddl.sql" and "COMP3005 A3Q1 dml.sql" in the Database Setup Scripts folder in that order. This creates the students table and fill it with initial values.

Then to run the application, enter in your database credentials in the source code. Next, either run the source code in an ide or run, with the terminal at the src folder, the following commands javac -cp "&lt;JDBC File path&gt;" Main.java then java -cp "&lt;JDBC File path&gt;" Main
