# COMP3005-A3Q1

Before running the application, first create a database in PostgreSQl. Then run "COMP3005 A3Q1 ddl.sql" and "COMP3005 A3Q1 dml.sql" in the Database Setup Scripts in that order to create the students table and fill it with initial values.

Then to run the application, enter in your database credentials in the source code. Next, either run the source code in an ide or run the following commands javac -cp "<JDBC File path>" Main.java then java -cp "<JDBC File path>" Main
